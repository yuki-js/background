# 雙峰祭の背景アニメーション

## なにこれ

はいどうも〜！バーチャル女の子、筑波のキズナアイこと、つくコレ ミスNo.2の青木ゆうきです！

雙峰祭2019前夜祭でスクリーンに投影したグラフィックのソースコードのOSS版です！

それでは早速〜ビルド行ってみよう！

なお、OSS化につき一部変更しています。

## ビルド

ビルド環境: macOS Mojave
他プラットフォームは
```
- #include <GLUT/glut.h>
+ #include <GL/glut.h>
```
するなど

```
$ ./ai
```
